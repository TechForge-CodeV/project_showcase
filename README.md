# Code-V 🚀

A modern, responsive **Developer Club Showcase Website** built to highlight projects, team members, and collaboration opportunities.

---

## 📌 Overview

**Code-V** is a frontend web project designed to:

* Showcase developer projects
* Display team members
* Enable collaboration opportunities
* Provide a clean and modern UI/UX experience

This project acts as a **portfolio + community hub** for a developer club.

---

## ✨ Features

* 🌙 Dark/Light theme toggle
* 📱 Fully responsive design (mobile-friendly)
* 🧑‍💻 Developer/member showcase
* 📂 Project listing section
* 🔍 Clean navigation with modal menu (for mobile)
* 🎨 Modern glassmorphism UI design
* 🔐 Login modal (UI-based)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Custom + Glass UI)
* **JavaScript (Vanilla JS)** – Interactivity
* **Bootstrap 5** – Layout & components
* **Bootstrap Icons** – Icons

---

## 📁 Project Structure

```
Code-V/
│
├── index.html            # Main landing page
├── projects.html         # Projects showcase page
├── developers.html       # Developer/member page
├── about_us.html         # About section
│
├── style.css             # Main styling file
├── app.js                # Core JavaScript logic
│
├── project-data.js       # Project data (dynamic rendering)
├── developers-data.js    # Developer data (dynamic rendering)
│
└── README.md             # Project documentation
```

---

## ⚙️ How It Works

* Project and developer details are stored in separate **JavaScript data files**.
* These are dynamically rendered into the UI using JavaScript.
* Bootstrap is used for layout, responsiveness, and components like modals.
* A theme toggle button switches between dark and light modes.
